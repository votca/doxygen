36  /* Calculating the AO matrix of the gradient operator requires
37  * the raw overlap matrix (i.e. in unnormalized cartesians)
38  * with lmax of the column shell increased by one:
39  *
40  * phi_(ijk) = x^i y^j z^k exp(-beta r^2)
41  * => d/dx phi_(ijk) = (i*x^(i-1) - 2beta x^(i+1)) y^j z^k exp(-beta r^2)
42  * d/dy phi_(ijk) = x^i (j*y^(j-1) - 2beta y^(j+1)) z^k exp(-beta r^2)
43  * d/dz phi_(ijk) = x^i y^j (k*z^(k-1) - 2beta z^(k+1)) exp(-beta r^2)
44  *
45  * i.e.: d/dx phi_s = d/dx phi_(000) = -2beta phi_(100) = -2beta phi_px
46  * d/dy phi_px = d/dy phi_(100) = -2beta phi_(110) = -2beta phi_dxy
47  * d/dz phi_pz = d/dz phi_(001) = phi_(000) - 2beta phi_(002)
48  * = phi_s - 2beta phi_dxx
49  *
50  * and with that
51  * <s|d/dx|s> = -2beta <s|px>
52  * <s|d/dy|px> = -2beta <s|dxy>
53  * <s|d/dz|pz> = <s|s> - 2beta <s|dxx>
54  * ...
55  *
56  */
