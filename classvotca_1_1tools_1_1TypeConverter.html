Detailed Description

template<class T, class G>
class votca::tools::TypeConverter< T, G >

Provides a means to reduce cross dependency of header files.

This templace class allows the conversion of one type of object to another this is done by breaking the said object up into the standard types.

For the type converter to work each object it is converting between must have defined at least one write function and or one read function.

This tempalte object only allows converting between doubles,strings,ints.
