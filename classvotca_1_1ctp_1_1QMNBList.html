void votca::ctp::QMNBList::GenerateSuperExchange ( )

Adds SuperExchange pairs to the neighbor list.

SuperExchange pairs are those pairs which have one or more bridging molecules specified in the input file via a record [Donor Bridge1 Bridge2 ... Acceptor] Every pair gets a flag, identifying if a QMPair is of type Hopping, SuperExchange, SuperExchangeAndHopping. This is stored to the state file The BRIDGED pairs are stored but BRIDGING pairs have to be regenerated every time we need them (edft job writer, idft job writer and importer)
