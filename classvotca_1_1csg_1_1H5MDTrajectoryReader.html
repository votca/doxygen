Detailed Description

class for reading H5MD trajectory.

This class implements the H5MD trajectory reading function. The format of the H5MD file is defined in Pierre de Buyl, Peter H. Colberg, Felix Höfling, H5MD: A structured, efficient, and portable file format for molecular data, http://dx.doi.org/10.1016/j.cpc.2014.01.018.

The current reference is available here: http://nongnu.org/h5md/
