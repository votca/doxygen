BOOST_AUTO_TEST_CASE ( test_trajectoryreader  )

Test the trajectory reader.

This test is designed to test the trajectory reader this is done by creating a small lammps dump file. A topology object is created with some default values. The file is then read in with the trajectory reader and the values in the top object are then examined to ensure they no longer represent the default state but the values from the file.
