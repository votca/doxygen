17 // NOTE: This is not a conventional 3D Ewald summation, so use carefully
18 // (tuned for the purpose of cluster energy calculations)
19 // NOTE: PolarTop should be set-up with three containers: FGC, FGN, BGN
20 // MGN is set-up in constructor using the real-space c/o (from input)
21 // The topology is used to retrieve information on the sim. box (PB)
22 // All polar segments should be positioned as nearest images of the
23 // foreground charge density (FGC, FGN).
24 // All polar segments should be appropriately charged (Q00, Q10, ...).
25 // NOTE: The k-shell grouping algorithm can fail for strongly skewed boxes.
26 //
