Detailed Description

Base class for all calculators.

Calculators are grouped in CalculatorFactories and are run by Threads or Applications. Every calculator has a description (an XML file) installed in VOTCASHARE which is used to compile HELP and run TESTSUITE. This XML file also contains default values.
